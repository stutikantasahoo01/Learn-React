# To install axios 
# npm i axios
# 1. What is Axios?
# Axios is a popular third-party JavaScript library used to make HTTP requests (like GET or POST) from the browser or Node.js. It is built on top of the native Promises API.

# Key Feature: It automatically transforms data into JSON.

# Key Feature: It has built-in protection against XSRF (Cross-Site Request Forgery).

# Key Feature: It works in both the browser and Node.js environments.
